硬件驱动与简单调度系统实现是嵌入式应用的重要领域，
% 嵌入式应用
现代电梯系统是由计算机操控的，
在这里计算机的工作是接受信息并且正确的调度操控电机工作，
并完成相关显示功能，
系统需要知道电梯在哪里，乘客在哪里，
乘客想要去哪里，
得到这些信息，
电梯系统需要通过电梯调度完成需求
\upcite{patjoshi2010design}.


随着电梯在应用中使用越来越广泛，
多电梯系统成为趋势
\upcite{liew2015review},
对电梯调度算法的了解分析越来越重重要.
本项目利用现有的资源，
对电梯系统的工作做出简单模拟分析。
